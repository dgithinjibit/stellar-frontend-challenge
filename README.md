# 🚀 Stellar White Belt: Payment dApp

A functional Stellar decentralized application (dApp) built for the **Stellar White Belt (Level 1)** submission. This project demonstrates core proficiency in wallet integration, balance retrieval, and transaction execution on the Stellar Testnet.

## 🎯 Submission Requirements Checklist
- [x] **Wallet Setup**: Configured for Freighter Wallet on Stellar Testnet.
- [x] **Connection**: Robust Wallet Connect/Disconnect functionality.
- [x] **Balance**: Real-time XLM balance fetching and UI display.
- [x] **Transaction**: Send XLM functionality with transaction hash feedback.
- [x] **Deployment**: Publicly hosted and open-source repository.
